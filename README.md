# Eventix - Modern Event & Conference Website

**Eventix** is a modern, responsive landing page concept for an event and conference website. The design focuses on clarity, professionalism, and an engaging hero section layout with side-by-side image and content alignment.

This project is a static front-end replica inspired by the Eventix demo design. It was built using **HTML5** and **CSS3**, with **Bootstrap Icons** integrated for modern, lightweight iconography.

## Features

- Clean and responsive design layout  
- Sticky navigation bar with dropdown-ready menu items  
- Side-by-side hero section (text + image)  
- Floating feature buttons beside the main image  
- Consistent font and color scheme based on the Eventix theme  
- Google Fonts integration (Poppins)  
- Reusable CSS variables for color and typography management  

---

## Technologies Used

- **HTML5** – Page structure and content layout  
- **CSS3** – Custom styling and responsive design  
- **Bootstrap Icons** – For vector-based, lightweight icons  
- **Google Fonts** – Poppins font family for modern and clean typography
